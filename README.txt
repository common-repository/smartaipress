== Installation ==

The installation of the SmartAIPress plugin is straight forward and very easy.
You can install it from the WordPress admin dashboard plugins page, 
or you can manually install it by downloading the smartaipress.zip file from wp.org and follow next procedure:

1. Unzip the smartaipress.zip file.
2. Upload `smartaipress` folder to the `/wp-content/plugins/` directory
3. Activate the plugin through the 'Plugins' menu in WordPress
